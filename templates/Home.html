Overview


During the COVID-19 Pandemic, many small brick and mortar stores have closed their doors forever. This website was created to show tourists, and local residents of the State of New Jersey some better alternatives to other bigger box stores.


Team Python


This website was created for the CPS 4951 Senior Capstone Class at Kean University. The website was made using the Flask Library in Python. The main reason we selected Flask, is because it allows us to insert special blocks of code into HTML, allowing us to use Python alongside HTML. We also used Python, JavaScript, CSS, HTML, and MySQL. The two creators of this website were Kevin Asaro and Omar Hewedy. Omar was Front End, while Kevin was Back End.
